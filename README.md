🔥 Habit Streak Visualizer
A beautiful, production-ready web application to build better habits through daily tracking and streak visualization. Track your progress, maintain consistency, and watch your streaks grow!
✨ Features
📊 Smart Habit Tracking

--> Daily Completion Tracking - Mark habits as complete with a single click
--> Streak Calculation - Automatic tracking of current and longest streaks
--> Multi-Habit Support - Track unlimited habits simultaneously
--> Habit Management - Easy create, view, and delete operations

📈 Beautiful Analytics Dashboard

Real-time Stats Cards - Total habits, longest streak, completion rate, and consistency score
Interactive Charts - 4 different chart types powered by Chart.js:

-->Line Chart: 30-day streak growth visualization
-->Bar Chart: Weekly completion breakdown
-->Pie Chart: Completed vs. missed habits ratio
-->Heatmap: GitHub-style 90-day activity visualization

🎨 Modern UI/UX

-->Dark/Light Mode - Toggle between themes with automatic preference saving
-->Responsive Design - Seamless experience across desktop, tablet, and mobile
-->Beautiful Gradients - Eye-catching purple and teal color schemes
-->Smooth Animations - Polished transitions and micro-interactions
-->Toast Notifications - Instant feedback for user actions

🔐 Secure Authentication

-->HTTP Basic Authentication - Simple, reliable user authentication
-->Multi-User Support - Each user has their own private habit data
-->Secure Password Storage - bcrypt password hashing
-->Auto-Login - Remember Me functionality for convenience

🛠️ Built With

Backend
-->Flask 3.0 - Lightweight Python web framework
-->SQLAlchemy 2.0 - Modern Python SQL toolkit and ORM
-->MySQL 8.0 - Reliable relational database
-->bcrypt - Secure password hashing
-->Flask-CORS - Cross-Origin Resource Sharing support

Frontend
-->HTML5 - Semantic structure
-->Tailwind CSS - Utility-first CSS framework
-->Vanilla JavaScript - No dependencies, pure ES6+
-->Chart.js - Beautiful, responsive charts
-->Custom CSS - Hand-crafted animations and gradients
