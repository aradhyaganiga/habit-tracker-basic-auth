# 🔥 Habit Streak Visualizer

A **beautiful, production-ready web application** designed to help users build better habits through **daily tracking, streak visualization, and insightful analytics**.  
Track your progress, maintain consistency, and watch your streaks grow 📈

---

## ✨ Features

### 📊 Smart Habit Tracking
- ✅ **Daily Completion Tracking** – Mark habits as complete with a single click  
- 🔁 **Automatic Streak Calculation** – Tracks current and longest streaks  
- 📚 **Multi-Habit Support** – Track unlimited habits simultaneously  
- 🗂️ **Habit Management** – Create, view, and delete habits easily  

---

### 📈 Beautiful Analytics Dashboard
- 📌 **Real-Time Stats Cards**
  - Total habits  
  - Longest streak  
  - Completion rate  
  - Consistency score  

- 📊 **Interactive Charts (powered by Chart.js)**
  - 📈 **Line Chart** – 30-day streak growth visualization  
  - 📊 **Bar Chart** – Weekly habit completion breakdown  
  - 🥧 **Pie Chart** – Completed vs missed habits ratio  
  - 🔥 **Heatmap** – GitHub-style 90-day activity visualization  

---

### 🎨 Modern UI / UX
- 🌗 **Dark / Light Mode** – Theme toggle with saved user preference  
- 📱 **Responsive Design** – Optimized for desktop, tablet, and mobile  
- 🎨 **Beautiful Gradients** – Purple & teal color palette  
- ✨ **Smooth Animations** – Polished transitions and micro-interactions  
- 🔔 **Toast Notifications** – Instant feedback for user actions  

---

### 🔐 Secure Authentication
- 🔑 **HTTP Basic Authentication** – Simple and reliable auth mechanism  
- 👥 **Multi-User Support** – Each user has private habit data  
- 🔒 **Secure Password Storage** – Passwords hashed using `bcrypt`  
- 💾 **Auto-Login** – “Remember Me” functionality for convenience  

---

## 🛠️ Tech Stack

### Backend
- **Flask 3.0** – Lightweight Python web framework  
- **SQLAlchemy 2.0** – Modern ORM for database interaction  
- **MySQL 8.0** – Relational database for persistent storage  
- **bcrypt** – Secure password hashing  
- **Flask-CORS** – Cross-Origin Resource Sharing support  

### Frontend
- **HTML5** – Semantic and accessible structure  
- **Tailwind CSS** – Utility-first styling framework  
- **Vanilla JavaScript (ES6+)** – Lightweight, dependency-free logic  
- **Chart.js** – Interactive and responsive charts  
- **Custom CSS** – Hand-crafted animations and gradients
